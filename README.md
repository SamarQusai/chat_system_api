# About Chat System
* Simple endpoint to create chat for spceific application.

Built With
------------
* **Golang**

Requirements
------------
* Golang

How to run
--------
```bash
$ git clone git@github.com:SamarQusai/chat_system_api.git
$ cd chat_system_api/
$ go get
$ go run main.go routes.go
```
